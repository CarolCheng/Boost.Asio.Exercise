# Boost.Asio.Exercise
##Environment
1. Ubuntu 14.04.5 LTS
  * Command to check version : *lsb_release -a*
2. C++11
3. Boost.Asio 1.59.0 
  * Dependencies: 
     * Boost.System
     * Boost.Regex
     * OpenSSL

##Content
1. A TCP Echo Client and Server
2. A synchronous TCP Client and Server
3. A asynchronous TCP Client and Server
